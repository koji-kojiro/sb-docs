## Variable: ***sb-ext:*enclosing-source-cutoff****

### Value
```
1
```
### Documentation
```
The maximum number of enclosing non-original source forms (i.e. from
  macroexpansion) that we print in full. For additional enclosing forms, we
  print only the CAR.
```
