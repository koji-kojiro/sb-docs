## Macro: ***sb-alien:cast***

### Lambda list
```
(ALIEN TYPE)
```
### Documentation
```
Convert ALIEN to an Alien of the specified TYPE (not evaluated.)  Both types
   must be Alien array, pointer or function types.
```
