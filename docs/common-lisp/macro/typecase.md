## Macro: COMMON-LISP:TYPECASE

> TYPECASE Keyform {(Type Form*)}*
> Evaluates the Forms in the first clause for which TYPEP of Keyform and Type
> is true.

### Lambda list
```
(KEYFORM &BODY CASES)
```
