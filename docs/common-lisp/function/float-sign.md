### Lambda list
```
(FLOAT1 &OPTIONAL FLOAT2)
```
### Documentation
```
Return a floating-point number that has the same sign as
   FLOAT1 and, if FLOAT2 is given, has the same absolute value
   as FLOAT2.
```
